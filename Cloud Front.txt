1. Open EC2, Create ec2 server No. 1 in us-east-1a
type :- AWS AMI
VPC :- default
key pair :- create new key pair PEM
security group :- 
	1. SSH - anywhere
	2. HTTP - anywhere

2. add inbound and outbound all anywhere for both server.

3. Run following in server
sudo su
yum update -y 
yum install -y httpd
systemctl start httpd
chkconfig httpd on
yum install -y php8.2

4. run "sudo nano /etc/httpd/conf/httpd.conf"
edit <ifModule> tag 
add "index.php" beside "index.html"
press ctrl+o and Enter
press ctrl+x

5. Run following command :-
sudo service httpd restart
chmod 777 /var/www/html
cd /var/www/html

6. run "vi index.php"
add following content in index.php
<?php
date_default_timezone_set('Asia/Kolkata');
echo "Instance 1:".date("d-m-y h:i:sa");
?>

7. open Cloud front, create distribution
origin domain :- Public IPv4 DNS of created EC2 with "http", ex:- http://ec2-54-84-85-35.compute-1.amazonaws.com
protocol :- HTTP
Enable Origin Shield :- True
Origin Shield region :- Same as EC2 region
Legacy cache settings :- True
Headers/Query strings/Cookies :- All
Response headers policy - optional :- CORS-with-preflight-and-securityHeadersPolicy
Web Application Firewall (WAF) :- Do not enable security protections
